# Telegram Mailing Bot

Telegram-бот для рассылок, публикации постов и планирования сообщений.

## Описание

Проект представляет собой Telegram-бота, позволяющего управлять рассылками, создавать отложенные публикации и взаимодействовать с пользователями через Telegram API.

Бот поддерживает систему подписок, планирование постов и набор административных команд для управления контентом и статистикой.

## Возможности

- Рассылка сообщений подписчикам
- Создание и публикация постов
- Планирование отложенных сообщений
- Система подписок
- Административные команды
- Просмотр статистики

## Используемые технологии

- Python
- Telegram Bot API
- SQLite / PostgreSQL
- python-dotenv
